# FinanceHub - Personal Finance Dashboard

A modern, beautiful web app to track your expenses, visualize spending patterns, and manage your budget with ease.

## Features

✨ **Core Functionality**
- 💰 Track expenses with categories, amounts, dates, and descriptions
- 📊 Visual dashboard with real-time stats (Budget, Spent, Remaining)
- 📈 Interactive charts - Pie chart and Bar chart breakdowns by category
- 🎯 Budget management - Set and monitor your budget usage
- 🗑️ Easy expense management - Add and delete expenses
- 📱 Fully responsive design - Works perfectly on mobile, tablet, and desktop

## Tech Stack

- **React 18** - Modern UI library
- **Vite** - Lightning-fast build tool
- **Recharts** - Beautiful data visualization
- **Lucide React** - Crisp icon library
- **CSS3** - Modern styling with gradients and animations

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The app will open in your browser at `http://localhost:5173`

## Usage

1. **Add Expenses** - Fill in the expense form with amount, category, date, and description
2. **View Dashboard** - See real-time stats showing budget, spent, and remaining amounts
3. **Visualize Spending** - Check pie and bar charts to understand spending patterns
4. **Manage Budget** - Update your budget limit anytime
5. **Delete Expenses** - Remove incorrect or unwanted expense entries
